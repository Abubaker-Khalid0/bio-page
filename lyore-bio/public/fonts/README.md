# Font Files

This directory contains self-hosted font files for the LYORE ABAYA bio page.

## Required Fonts

Download the following fonts in `.woff2` format and place them in their respective directories:

### 1. Playfair Display
- **Directory**: `playfair-display/`
- **File**: `playfair-display.woff2`
- **Weights**: 400, 600, 700
- **Source**: [Google Fonts](https://fonts.google.com/specimen/Playfair+Display)
- **Usage**: English headings and brand name

### 2. Inter
- **Directory**: `inter/`
- **File**: `inter.woff2`
- **Weights**: 300, 400, 600
- **Source**: [Google Fonts](https://fonts.google.com/specimen/Inter)
- **Usage**: English body text

### 3. Noto Naskh Arabic
- **Directory**: `noto-naskh-arabic/`
- **File**: `noto-naskh-arabic.woff2`
- **Weights**: 400, 600, 700
- **Source**: [Google Fonts](https://fonts.google.com/noto/specimen/Noto+Naskh+Arabic)
- **Usage**: Arabic headings

### 4. Tajawal
- **Directory**: `tajawal/`
- **File**: `tajawal.woff2`
- **Weights**: 300, 400, 500
- **Source**: [Google Fonts](https://fonts.google.com/specimen/Tajawal)
- **Usage**: Arabic body text

## Download Instructions

1. Visit Google Fonts for each font
2. Select the required weights
3. Download as `.woff2` format (preferred for web performance)
4. Rename files to match the structure above
5. Place in the correct directory

## File Size Guidelines

- Each font file should be under 100KB
- Use variable fonts when available for better performance
- Subset fonts to include only required characters if possible
